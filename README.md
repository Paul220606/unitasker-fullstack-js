# Unitasker

A full-stack JavaScript task management application.

## Features
- User registration & login
- Create, update, delete daily tasks
- Mark tasks as completed
- Secure API with role-based access
- Responsive UI

## Tech Stack
### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express
- MongoDB
